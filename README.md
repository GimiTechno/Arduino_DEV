# 【Arduino(公開:開発用)】
 
約10年、開発してきたArduino(AVRマイコン)系の開発をSVNからGitに以降。
その公開開発リポジトリ。組み込み屋として常にコーディングして食べてます。
 
## 【機能(ドライバ等)】
 
・CLK ... None
 
・WDT ... None
 
・Timer ... None
 
・UART ... デバッグモニタ
 
・CAN ... クラシックCAN 
 
・SPI ... MCP2515(CANコントローラ)、MCP25625（CANコントローラ、トランシーバ）ドライバ
 
・I2C ... SHT31(湿温度センサ)
 
・ADC ... None
